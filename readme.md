# NewQRcode 新二维码生成器
##

###开源状态

- 应用名称：	新二维码生成器
- 开发者名称：赵树豪
- 应用介绍：	新二维码生成器!全新架构，全新界面，全新体验,移动端首创图片生成二维码、音频二维码、软件二维码,欢迎大家体验! 
- 版本号：36
- Target Version： 22（即将支持23）

###软件架构：MVP

- model：逻辑层，处理主要业务逻辑
- view：包含应用的主要Activity
- present：中间处理层
- bean：云端表结构

###创新源泉

>受草料二维码启发，移动端首创图片二维码、音频二维码、文件二维码功能，可以自由的将图片、音频、文件生成二维码进行扫描，其余功能为常用二维码生成功能
